工具介绍

在传统的历史地理研究中，我们常常面临一个棘手的痛点：历史地图和早期卫星图中蕴藏着极具价值的地名与地理标注，但由于它们仅仅是静态图像，想要在海量图库中寻找某个特定的历史节点，往往只能在一张张地图中肉眼检索，费时费力。

另一方面，如果我们想让这些古旧地图重获空间属性，与现代地图进行位置匹配，传统做法必须依赖庞大且复杂的桌面端 GIS 软件。繁琐的安装和学习过程，对初学者或跨领域的文史研究者来说极不友好。

基于此，我们设计了这套轻量级的在线工具集，旨在为你提供一条从“简单易用”到“专业处理”的无缝互通工作流：

  • 地图标注工具： 将地图中沉睡的文字提取出来。支持在超大分辨率的历史原图上直接进行文本标注，将不可读的图像像素转化为支持全局模糊检索的结构化数据，让文献查找告别“海里捞针”。
  • 在线配准工具： 彻底打破软件壁垒。在浏览器内即可快速完成历史地图与现代底图的坐标对齐。生成的参数与影像（TFW / GeoTIFF）完全兼容专业 GIS 软件。

这两个工具不仅可以独立工作，更能够相互赋能。当你在配准工具中完成地图的位置校准后，可以直接导入标注工具生成的 JSON 标注文件。系统底层会利用仿射变换矩阵，将原先基于图片像素定位的标注，一键转换为带有标准经纬度坐标的 GeoJSON 地理空间数据。这意味着，你在网页上轻松完成的每一次打点，都能彻底融入标准的 GIS 系统与数据库中。

地图标注工具

让古地图的文字重新活过来

1. 简洁布局与功能分区

界面设计遵循极简与高效原则。核心区域划分为：顶部导航栏（控制全局导入导出）、全屏地图画布（沉浸式查看与交互）、以及右侧边栏（集成搜索、标注样式调整与列表管理）。布局逻辑清晰，让研究者告别繁杂的软件面板，专注于史料本身的解读与数据提取。

2. 云端地图一键导入

系统深度集成了云端共享图库，一键即可加载远端地图。底层采用了“混合金字塔渲染技术”（Deep Zoom），无论多大分辨率（>500MB）的历史扫描件或 GeoTIFF，都能在浏览器中实现浏览与平滑缩放。同时支持“跨图标注搜索”，帮你快速定位不同地图上的同一地名。

3. 可自定义属性标注

针对历史方志和古旧地图复杂的排版习惯，工具提供了高度自由的标注样式调节。你可以随时调整文字的大小、字距，甚至一键切换横排与竖排方向。配合丰富的快捷键支持（如 Shift 旋转、中括号调字距等），让海量地名的录入工作变得高效且符合直觉。

4. 搜索定位快速查阅

告别肉眼寻找。你添加的每一个标注都会被系统实时建立索引。内置的搜索引擎支持“模糊匹配”和“别名检索”。只需在搜索框输入地名残片，系统即可在列表中瞬间反馈匹配结果，点击即可控制地图平滑移动并聚焦到该标注的精确位置。

5. 像素定位与 JSON 存储

你的所有工作成果绝不会被困在图像里。每一次打点，系统都会基于地图的高精度像素坐标进行独立记录。这些数据被组织成轻量、易读的 JSON 结构化文件（包含 ID、文本、像素 X/Y 等）。你可以随时将其导出为本地文件，不仅方便后续修改还原，更为之后的数据升维（如转换 GeoJSON）打下坚实基础。

在线地理配准工具

连接历史地图与现代坐标系

1. QGIS同款布局分区

界面设计深度参考了 QGIS 等专业桌面 GIS 软件的经典布局。左侧为待配准的历史原图，右侧为接入了天地图的现代参照底图，底部则是直观的控制点（GCPs）管理面板。双屏联动的同屏打点方式，不仅让专业用户得心应手，也极大降低了初学者和跨领域学者的学习成本。

2. 纯前端超大图浏览

彻底打破传统 Web GIS 对服务器切片引擎的依赖。凭借 OpenSeadragon 框架与 GeoTIFF.js 的深度融合，工具在纯浏览器环境下即可利用“动态混合金字塔切片”技术，流畅加载并渲染动辄数 GB 的超大分辨率高清历史图像，全程无需上传服务器，保障数据隐私与极致速度。

3. 配准地图叠加预览

提供极速的配准视觉反馈。当你打下足够的控制点后，系统底层会运用最小二乘法算法，在现代底图上渲染出历史地图的高清叠加层。系统支持从一阶仿射到三阶多项式等多种空间变换模型，并实时计算展示每个控制点的 RMS 误差，助你精准校对每一寸地理细节。

4. 数据互通与软件兼容

坚持开放的数据标准，绝不“绑架”你的研究成果。你可以在此完成轻量化的前期工作，随后将控制点位（TXT/CSV）、空间坐标参考文件（TFW），甚至压制好地理坐标的完整 GeoTIFF 影像一键下载至本地。所有产出均与 ArcGIS、QGIS、奥维互动地图等主流专业软件完美兼容。

5. 工具联动与坐标转换

这是工作流中最具价值的一环。完成地图配准后，你可以直接导入“地图标注工具”生成的独立像素 JSON 文件。系统会利用当前解算出的仿射变换矩阵，将原本只懂“X/Y 像素位置”的纯文本标注，一键升维转换成带有标准经纬度坐标的 GeoJSON 地理矢量数据，彻底将其接入现代空间数据库。
